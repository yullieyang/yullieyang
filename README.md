## What I'm focused on

Production-style research-support code: pulling macro / trade / commodity series from authoritative sources (FRED), harmonizing across frequencies, computing derived measures, and producing short-horizon forecasts with documented uncertainty — packaged so a reviewer can audit every step from Git.
